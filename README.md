# Portfolio Website

A modern portfolio website built with React and Tailwind CSS.

## Features
- Responsive design
- Modern UI/UX
- Smooth animations
- Skills showcase
- Project gallery

## Technologies Used
- React
- Tailwind CSS
- Framer Motion
- TypeScript

## Getting Started
1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/REPO_NAME.git
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```
